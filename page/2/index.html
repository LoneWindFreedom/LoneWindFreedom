Ajax请求

前台：
1.使用JavaScript：

<script type="text/javascript">
    var xmlHttp;
    //获取请求对象
    function getXmlHttpRequest(){
    if (window.ActiveXObject) {//如果是IE浏览器则创建一个ActiveXObject("Microsoft.XMLHTTP")
        //创建一个ActiveXObject
        xmlHttp = new ActiveXObject("Microsoft.XMLHTTP");
        return xmlHttp;
    } else if (window.XMLHttpRequest) { //非ie浏览器
        //创建一个该对象
        xmlHttp = new XMLHttpRequest();
        return xmlHttp;
    }
    return null;
    }
    //get模式请求
    function doGet(){
        //1.获取请求对象
        xmlHttp=getXmlHttpRequest();
        if (xmlHttp == null) {
        return;
        }
        //2.设置请求地址
        var url="AjaxDemo/javascrip";
        xmlHttp.open("post", url,true);
        //3.设置回调方法
        xmlHttp.onreadystatechange = function(){
        //3.1 判断状态为4(处理完成)和http响应码为200(成立成功)再做处理,
        if (xmlHttp.readyState == 4 && xmlHttp.status == 200) {
            //3.2 获取响应的结果文本
            var data = xmlHttp.responseText;
            alert(data);
            }
        }
        xmlHttp.setRequestHeader("Content-type","application/x-www-form-urlencoded");
        xmlHttp.send("userAge=28");//userAge表示后台@RequestParam 修饰的userAge参数
    }
</script>

后台：

@ResponseBody
@RequestMapping(value="javascrip")
public String   ajax(@RequestParam Integer userAge){
    System.out.println("userAge"+userAge);
    return "成功";
}

@ResponseBody
@RequestMapping(value="JqueryGet")//produces = "application/json;charset=UTF-8"表示返回一个json对象
public String ajaxJquery(@RequestParam String userName){
    System.out.println(userName);
    return "请求成功";
}
@ResponseBody
@RequestMapping(value="JqueryPost",produces = "application/json;charset=UTF-8")//produces="text/html;charset=UTF-8"表示返回的数据为字符串格式
public Map<String, String> ajaxPost(@RequestParam String userName,Map<String, String> map){
    System.out.println(userName);
    map.put("msg", "成功");
    return map;
}


注：ajax返回Map对象或其它集合时，需要使用Jackson-core-asl.jar和Jackson-mapper-asl.jar,可以在springMvc配置文件中注解驱动标签中设置；返回字符串时，需要fastjson.jar来处理编码格式问题，也可以设置@RequestMapping注解中的produces属性为”text/html;charset=utf-8”。

排序算法

  1. 冒泡排序
    原理：取相邻位置元素按照从大到小或者从小到大比较，并交换位置。
int[] array = {63,4,24,1,3,13};//初始化数组 
for (int i = 1; i < array.length; i++) {  //排序
    for (int j = 0; j < array.length - i; j++) {
        if(array[j] > array[j+1]){
           int temp = array[j];
           array[j] = array[j+1];
           array[j+1] = temp;  
        }
    }
}

2.选择排序
原理：依次取所有位置的元素与未排序的元素按照一定规则比较，并交换位置。

public static void select(){
    // 初始化数组
    int[] array = { 63, 4, 24, 1, 3, 13 };
    // 排序
    int len = array.length;
    // 控制轮数
    for (int i = 1; i < len; i++) {
        int max = array[0];
        int index = 0;
        // 查找最大值
        for (int j = 1; j < len - (i - 1); j++) {
            if (max < array[j]) {
                max = array[j];
                index = j;
            }
        }
        // 互换位置
        int temp = array[index];
        array[index] = array[len - i];
        array[len - i] = temp;
    }
    //输出
    System.out.println(Arrays.toString(array));
}

3.插入排序
原理：取所有位置的元素与已排序的元素按照一定规则比较，并找出要插入的位置，移动相关元素及插入元素。

//初始化数组
int[] array = {20,40,90,30,80,70,50};
//排序
int j;
for (int i = 1; i < array.length; i++) {
    int temp = array[i];
    for (j = i - 1; j > 0 && array[j] > temp; j--) {
        array[j+1] = array[j];
     }
     array[j+1] = temp;
}

简单搭建

简单搭建：
1.环境搭建
1.1安装Git
1.2安装node.js
2.配置GitHub
2.1创建Respository库，名称为github用户名.github.io
2.2安装git-bash，打开后生成公钥：ssh-keygen -t rsa -C “github邮箱地址”，将.ssh文件夹下的密钥内容复制到github设置中的ssh选项中。
3.安装Hexo
3.1
打开git命令行：
$ npm config set registry https://registry.npm.taobap.org
重启git命令行：
$ npm install -g hexo
3.2在任意盘符下新建一个空文件夹，如：D:\blog,右击选择git-bash
$ hexo init
[info] Copying data[info] You are almost done! Don’t forget to run npm install before you start blogging with Hexo!
3.3安装node_modules
$ npm install
4.启动服务
$ hexo server
[info] Hexo is running at http://localhost:4000/.Press Ctrl+C to stop.
4.2在浏览器地址栏输入：localhost:4000,检测是否成功启动
4.1 长按ctrl+c关闭服务
5.创建新的文章
$ hexo new “文章标题”
6.生成静态网页
$ hexo generate
6.1编辑网页内容
打开source文件加下的—post下对应的md文件，使用markdownpad编辑
7.将项目部署到github服务器
7.1配置
deploy:
type: git–>git3.0以上
repository:git@github.com:仓库名.github.io.git
branch: master
7.2部署器
$ npm install hexo-deployer-git –save
7.3部署
$ hexo clean 清理缓存
$ hexo generate
$ hexo deploy
或$ hexo deploy -g
