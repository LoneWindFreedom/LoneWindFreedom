简单搭建

简单搭建：
1.环境搭建
1.1安装Git
1.2安装node.js
2.配置GitHub
2.1创建Respository库，名称为github用户名.github.io
2.2安装git-bash，打开后生成公钥：ssh-keygen -t rsa -C “github邮箱地址”，将.ssh文件夹下的密钥内容复制到github设置中的ssh选项中。
3.安装Hexo
3.1
打开git命令行：
$ npm config set registry https://registry.npm.taobap.org
重启git命令行：
$ npm install -g hexo
3.2在任意盘符下新建一个空文件夹，如：D:\blog,右击选择git-bash
$ hexo init
[info] Copying data[info] You are almost done! Don’t forget to run npm install before you start blogging with Hexo!
3.3安装node_modules
$ npm install
4.启动服务
$ hexo server
[info] Hexo is running at http://localhost:4000/.Press Ctrl+C to stop.
4.2在浏览器地址栏输入：localhost:4000,检测是否成功启动
4.1 长按ctrl+c关闭服务
5.创建新的文章
$ hexo new “文章标题”
6.生成静态网页
$ hexo generate
6.1编辑网页内容
打开source文件加下的—post下对应的md文件，使用markdownpad编辑
7.将项目部署到github服务器
7.1配置
deploy:
type: git–>git3.0以上
repository:git@github.com:仓库名.github.io.git
branch: master
7.2部署器
$ npm install hexo-deployer-git –save
7.3部署
$ hexo clean 清理缓存
$ hexo generate
$ hexo deploy
或$ hexo deploy -g

操作说明

$ npm install -g hexo —>安装hexo
$ npm install hexo-deployer-git –save —>选择hexo-deployer-git
$ hexo init —>初始化hexo
$ npm install —>运行hexo
$ hexo server —>开启服务
press ctrl+c —>长按表示结束服务
$ hexo new “My New Post” —>创建一片新的文章
$ hexo clean —>清空
$ hexo generate —>将markdown文件生成静态网页!
$ hexo deploy —>部署到服务器
$ hexo g #本地生成静态网页
$ hexo s #启动本地服务，进行文章预览调试
$ hexo s -g 生成网页并部署
$ hexo new “postName” #新建文章
$ hexo new page “pageName” #新建页面
$ hexo generate #生成静态页面至public目录
$ hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
$ hexo deploy #将.deploy目录部署到GitHub
$ hexo help # 查看帮助
$ hexo version #查看Hexo的版本
$ hexo deploy -g #生成加部署
$ hexo server -g #生成加预览
$ git clone url –下载主题地址
$ git config –global –list —>查询配置信息
$ git config –global user.name “github邮箱地址” 设置邮箱地址
$ git config –global http.postBuffer 524288000 设置上传/下载文件大小
